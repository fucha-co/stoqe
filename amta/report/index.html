2. Methods.

All methods, data, and tools used for this report are benchmark industry-standards. This report analyses the objective performance of amta.org.au as used by visitors and Google. This report does not analyse subjective elements (visual design, Content Management Systems, etc.).
2.1
Website Traffic Data — Source: Google. Google Analytics has been recording amta.org.au's website traffic data since June 2007 making it the most comprehensive and reliable data set.

2.2
Website Data Period — 12 months comparative. All relative numbers in this report compare the 12 month data period ending March 31, 2017 to the 12 month data period ending March 31, 2016. The absolute dates are 01/04/2016-31/03/2017 v's 01/04/2015-31/03/2016.

2.3
Performance Testing Tools: Google PageSpeed, GTmetrix, WebPagetest, and Pingdom.

5. Conclusion.

The traffic data shows amta.org.au has significantly declined in visitors and Google traffic in the last 12 months; benchmark industry-standard testing tools validate amta.org.au as a "poor performing" website that "fails" multiple performance tests; critical content, performance, and accessibility issues are present in amta.org.au, all of which negatively impact usability, engagement, Google search results, and website traffic volumes.

No considerations for performance are present in the AMTA website.

In conclusion the AMTA website "fails" modern performance and quality standards. The data confirms that amta.org.au is not working effectively for AMTA, website visitors, Google, or social media: critically, the data confirms the current state of amta.org.au is working against all primary stakeholders. Without significant improvements in the quality of amta.org.au the user experience and engagement (18%) will remain poor and the traffic trends seen on amta.org.au in the last 12 months (-33%) will likely continue.

5. Recommendations.

Bring amta.org.au inline with current web standards
 The report is conclusive: the current AMTA website is a very low performing and very low quality website that does not meet performance or quality standards. In order for amta.org.au to work effectively for stakeholders it must at-least meet current web standards. This can be achieved by rebuilding the website from scratch or renovating the existing website.
