2. Methods.

All methods, data, and tools used for this report are benchmark industry-standards. This report analyses the objective performance of amta.org.au as used by visitors and Google. This report does not analyse subjective elements (visual design, Content Management Systems, etc.).
2.1
Data source by Google. Google Analytics is the benchmark industry-standard tool for website traffic reports. Google Analytics has been recording amta.org.au's data since June 2007 making it the most comprehensive and reliable data set.

2.2
Performance test by Google. Google provides a suite of free testing tools for web developers to accurately gauge website performance. PageSpeed Insight is one such tool which scores website performance and advises improvements.

2.3
Simulation test by WebPagetest. WebPagetest is an open source project that simulates a real visitor using your website and provides rich diagnostic reports in return.

2.4
Comparative Analysis Period = 12 months. All relative numbers in this report compare the 12 month data period ending March 31, 2017 to the 12 month data period ending March 31, 2016. The absolute dates are 01/04/2016-31/03/2017 v's 01/04/2015-31/03/2016.

5. Conclusion.

The AMTA website has significantly declined in all major performance metrics over the last 12 months. Benchmarch industry-standard testing tools validate amta.org.au as a very low quality website with critical performance, content, and accessibility issues. Such issues are negatively impacting usability, engagement, Google search results, website traffic volumes and, ultimately, AMTA.

The current AMTA website does not meet modern performance standards and does not present as a professional-level website. As of this report the current state of amta.org.au is very poor.

In conclusion amta.org.au is not working effectively for any primary stakeholders: critically, the current state of amta.org.au is working against all primary stakeholders and the decreasing KPI's are expected to continue.
