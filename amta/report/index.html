2. Method.

All methods, data, and tools used in this report are benchmark and industry-standard. All results are factual.
2.1
Data — Google Analytics. Google Analytics is the benchmark industry-standard tool for website traffic reports. Google Analytics has been recording data in amta.org.au since June 2007.

2.2
Performance — Google Developer Tools. Google provides a suite of testing tools for web developers to measure website performance and improvements.

2.3
Performance — webpagetest.org. Google Analytics is the benchmark industry-standard service for reporting website traffic metrics. Google Analytics has been recording data in amta.org.au since June 2007.

2.4
Comparative Analysis Period - 12 months. All relative numbers compare the 12 month data period ending March 2017 to the 12 month data period ending March 2016. The absolute dates are 01/04/2106-31/03/2017 v's 01/04/2015-31/03/2016.

3. Findings

3.1
Is the website working for AMTA? No. The number of visitors to amta.org.au has decreased 30.93% since April 2016 and February 2017 was the lowest month of website traffic since May 2011.

-30%

Total Visitors

Total visitors to amta.org.au have decreased 30.93% in 12 months from 533,283 to 368,314.

See the source data

-29%

New Visitors

New visitors to amta.org.au have decreased 29.22% in 12 months from 402,068 to 284,600.

See the source data

-36%

Return Visitors

Return visitors to amta.org.au have decreased 36.2% in 12 months from 131,215 to 83,714.

See the source data

3.2
Is the website working for visitors? No. The time visitors spend on amta.org.au has decreased 7.6% since April 2016. Bounce rates on amta.org.au are extreme: 88% of visitors leave amta.org.au without engaging (without clicking any links or visiting any other pages).

-7.6%

Time on Site

The average time visitors spends on amta.org.au has decreased 7.62% in 12 months from 46secs to 42secs.

See the source data

-9.9%

Time on Page

The average time visitors spends on a page has decreased 9.91% in 12 months from 3:27secs to 3:06secs.

See the source data

88%

Bounce Rate

The % of visitors who leave amta.org.au without any interaction (without clicking anything) is 88%.

See the source data
